# Skillify - Job & Skill Exchange Platform

Skillify is a full-stack web platform that connects job seekers, recruiters, and learners in one place.
It allows users to find jobs, hire talent, and exchange skills — making it more than just a traditional job portal.

This project is built with a focus on real-world functionality, scalability, and user experience, making it suitable for production-level use cases.
<br /><br /><br />

## Key Features
**1. User & Recruiter Roles**

  + Separate roles for Job Seekers and Recruiters
  + Role-based dashboard and functionality
  + Secure authentication system

**2. Job Portal System**

  + Recruiters can post, update, and delete jobs
  + Users can browse and apply for jobs
  + Admin panel to manage jobs and users

**3. Skill Exchange Platform**

  + Users can share skills and connect with others
  + Enables peer-to-peer learning

**4. Payment Integration (Razorpay)**

  + Integrated Razorpay Payment Gateway
  + Used for premium features / services
  + Secure and seamless payment flow

**5. Email Verification (PHPMailer)**

  + OTP-based email verification using PHPMailer
  + Prevents fake registrations
  + Improved authentication security

**6. Admin Panel**

  + Manage users, jobs, and platform content
  + Delete operations handled on the same page (no unnecessary redirects)
  + Clean and efficient UI

**7. Modern UI/UX**

  + Responsive design (mobile, tablet, desktop)
  + Interactive components like carousel sliders
  + Clean layout using modern CSS techniques
<br />

## Tech Stack

**1. Frontend**
  + HTML5
  + CSS3 (Flexbox, Responsive Design)
  + JavaScript
  + Bootstrap

**2. Backend**
  + PHP (Core PHP)
  + MySQL

**3. Integrations & Tools**
  + PHPMailer – Email OTP verification
  + Razorpay – Payment gateway
  + Git & GitHub
